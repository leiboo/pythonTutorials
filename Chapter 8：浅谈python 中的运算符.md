

# Chapter 8：浅谈python 中的运算符

## **逻辑运算符**

| 运算符 | 描述                                                         | 例子                   |
| ------ | ------------------------------------------------------------ | ---------------------- |
| and    | 所谓逻辑与运算符。如果两个操作数都为真，则条件为真。         | (a and b) 为 true.     |
| or     | 所谓逻辑OR运算符。如果有两个操作数都为非零，则条件变为真。   | (a or b) 为 true.      |
| not    | 所谓逻辑非运算符。用反转操作数的逻辑状态。如果条件为true，则逻辑非运算符将为false。 | not(a and b) 为 false. |
